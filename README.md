# Pune Air Quality Analysis and Prediction

## 📌 Overview
This project focuses on analyzing air quality data collected from the **Pune Kothrud AQMS station**. The dataset contains information about various pollutants and their respective Air Quality Index (AQI) values over time.

The work is divided into **two phases**:
- **Phase 1 (Current Phase):** Exploratory Data Analysis (EDA) – identifying trends, correlations, and insights from the data.

- **Phase 2 (Upcoming):** Development of a **Machine Learning model** to predict and forecast AQI levels based on pollutant concentrations.

---

## 📊 Phase 1: Exploratory Data Analysis
In this phase, the following tasks were carried out:
1. Data cleaning and preprocessing (handling missing values, formatting datetime columns, etc.)
2. Visualizations of pollutant concentrations and AQI trends.
3. Correlation analysis between pollutants and AQI.
4. Resampling and smoothing techniques to study daily and monthly AQI variations.
5. Identifying peak pollution periods and possible contributing factors.

---

## 🤖 Phase 2: Machine Learning (Work in Progress)
The next stage of this project involves:
- Feature engineering and selection of relevant pollutant metrics.
- Building supervised learning models for AQI prediction.
- Model evaluation and optimization for accuracy.
- Deploying the trained model for real-time or batch AQI forecasting.
